Proxy Guidelines

Requirements

  • Student's valid ID (with signature and picture)
  • Proxy's valid ID (with signature and picture)
  • Type-written or word-processed authorization letter (Sample Letter)
  • AISIS username / password of student (undergraduates only)

Authorization Letter Guidelines

  • Date letter was written
  • Date or duration of validity of the letrer
  • Name of proxy and authorization
  • Students signature over printed name

Other Reminders

  • One-to-one Policy for undergrauates (1 student 1 proxy)
  • Student and proxy can enlist at the same time when are of the same batch
  • Student proxy for School of Management must observe proper dress code
